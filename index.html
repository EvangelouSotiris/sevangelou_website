Asynxy: Asynchronous Proxy RCE

Asynxy is a "middleware" web application that performs remote code execution for a "master" node to a set of slaves. This is achieved through a periodic request for new commands from the slaves, a local execution using Python and lastly a post of the command's results back to the webapp for the master to see. The benefits that the webapp introduces is control over a large set of slaves either individually or as a group (botnet-like) and a connection mostly unbothered by firewalls or NAT since it is based on HTTP and is always initiated from the slaves.

Ingame Database App with Redis

A simple web application that uses the combination of Node.js and the Redis key-value store in order to simulate a "game" and the actions that an RPG player would use. The user can search for other users ,register a new account, login with his password, and do certain game actions such as looting an enemy, finding treasures, buying from stores, selling to stores and viewing his account information and inventory.
